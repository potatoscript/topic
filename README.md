### 医療分野のデジタルヘルスについて

```
医療分野は比較的景気の動向に左右されにくい分野とされていて、かつ昨今のコロナへの対応により、2022年は医療・健康領域のデジタル化が加速しました。

一か月ほど前のアメリカラスベガスで開催された展示市「CES」でも、デジタルヘルス関連は出展企業のカテゴリーとして一つの主要な分野になっています。

現在特に目立っているのは、遠隔医療システムなどのクラウドを基盤としたサービス、そしてそれを可能にするウェアラブルな生体データを測定するデバイスの開発です。

私は10年ほど前から、医療画像情報システムや医療機器の組込ソフトウェア開発の業務に携わることが多かったのですが、オンプレミスの医療情報サーバの保存データのレコード数が数億、数十億の単位と膨大になり、そのデータ処理が遅いということでパフォーマンス改善に四苦八苦していました。

ちょうどその頃遠隔診断サービスが検討され始めていた時期で、サーバのクラウド化が進み始めましたが、現状はまだクラウドに移行した医療機関は全体の数パーセントだそうです。

セキュリティやランニングコストなど課題が多いためだと思われますが、医療情報のセキュリティガイドラインは国が策定しており、最近では厚生労働省が主導となった電子処方箋管理サービスの運用が二週間ほど前先月1/26から開始されました。

システムを導入した医療機関や薬局はこのサービスを利用することで、電子処方箋の発行やそれに基づく調剤が可能になりました。

遠隔診療システムは、５Gの普及につれ、これから数年で一気に進むのではないかと予想されています。

遠隔医療サービスのうち、医師と医師の間の画像診断や病理診断サービスは進んでいますが、近年は、医師と患者の間の遠隔診療サービスも一般的になってきています。

それを可能にするのは、自宅で患者自身が操作可能な生体データを測定するデバイスで、その開発も活況を帯びています。

皮膚に貼って生体データ取得できるパッチ型のデバイスで聴診器やカメラ機能を持つデバイス

24時間生体データを記録し続ける充電不要のバッテリーを装備しているデバイスなど

スマホのカメラやマイクを使ってデータを取得するスマホアプリあるいはまだ実用化前ですが、トイレなどもスマートトイレとして、生体データを測定する機器として考えられているようです。

医療分野はそのほか、手術支援ロボットやＡＩ画像診断技術、臨床試験のDXなど、注目点はいろいろあり、従来の医療業界プレーヤーだけでなく、ＩＴ，電気、通信、保険業界など様々な企業が入り交じり、動きが活発化しています。

従来の思考や思い込みにとどまらない製品開発をしていくことが、ますます大事だと感じています。
```

### IT 人材の不足

- 世界の企業 IT 支出は 2023 年に 6.3％増へ、社内人材の不足が IT サービスの追い風に

```
一昨日の日経クロステックの記事に関して。世界の企業のITの支出が今年6.3%増ということで、社内の人材の不足がITサービスの追い風になるという記事があり紹介する。

アメリカのガードナーの最新の予測によると2023年はエンタープライズITと呼ばれる企業向けITの世界の支出額が、昨年に比べて6.3%増の2兆3825億ドル、309兆7215億円になる見通しである。

エンタープライズITと呼ばれているものは、データセンターシステム、サーバーやストレージ、ネットワーク機器、関連するシステムソフトを含める。

これにソフトウェアITサービスの３分野で、デバイスや通信サービスは含まれない。

ガードナーは、この全てのITの2023年の支出を2.4%増と予測しており、エンタープライズITの伸びを下回っている。各国の金利の引き上げや企業側の支出の引き締め、場合によっては人員削減をしているが、このエンタープライズITという支出については増やし続けているとのこと。

ITサービスの支出の増加についてガードナーは、企業の実装とサポートの為に外部スタッフの導入を検討していて、コンサルティングへの2023年の支出は6.7%増を見込もうとしている。

社内でITの人材を雇用する支出もあるが、ガートナーのCIO(最高情報責任者)は、IT人材の獲得競争に負けていて、外部のITサービスの支出はあらゆる業界で社内向けよりも急成長しているということで、スキルの高い人材は賃金要件や開発したいキャリアの向上についていけるテクノロジーサービスプロバイダへ移行しつつあるとコメントしている。

最後に、今年1月に6.3%増に予測が下がったが、激動時におけるCIOの意思決定も遅れがちではあるが、エンタープライズITの指数が不況に強いと見ている。コロナ禍に見舞われた2020年の伸び率を振り返ると、ガードナーは、4月に7.7%、7月に6.9%、10月には4.2%のマイナスの成長を予測し続けた。これは2020年の話だが、終わってみれば2020年の実績は4.1%増だったということで話を締めている。

この記事を読んで私が思ったのが、最近いわゆるGAFAに人材の削減のニュースがすごく多く取り沙汰されていること、指数の関係がどうなのかなっていうのがちょっと私もプロではないので分からないのですが、そういった見解もあるんだいうのが感じたところである。

これが日本の所まで波及しているかというところは分からないが。少くともわれわれメイテックではIT分野の専門の方がいて、そちらの分野に強いインフラ系の業務をされている方、あとソフトウェアの業務はもちろんだが、業務がまだまだ伸びていくという予測があるとしたら、ぜひここにわれわれも追いついて、しっかり乗っかってやって行ければという様に感じたので、紹介させてもらった。
```

口語(kougo)的表現を含めた文章プログラムをなります。

の知識を有しています。
教師学習なしの自立(jiritsu)学習式機能もあるので、
ユーザーとの受け答えによって徐々(jojo)に賢く(kashikoku)なっていくのが
可能で、次世代(jisedai)言語モデルとして、期待されています。

これまでも声音AIや画像認識AIなどが過去に登場していますが、ChatGPTは人の代わりに

GPT4となったときに、どのような精度・機能を併せ持ち(awasemochi)、
Googleのような検索システムに対抗するかが注目されています。

OpenAIは現時点ではチャットボットでしかありませんが、将来的には新しい検索エンジンに取って
代わる潜在能力を持ち、検索エンジンの中で大きな役割を担う(ninau)ことが可能であり、
名実(meijitsu)ともにGoogleの強敵(kyouteki)なライバルとなります。
ChatGPTの開発を受けて、すでにGoogleはAIの強化(kyouka)に
さらなる投資(toushi)をすることも宣言(sengen)しています。

### フードロスト

- 朝礼トピック

```
フードロストというと皆さん想像するのは、スーパーの惣菜やコンビニの弁当といった調理済みのもの、
売れ残ったものを廃棄するというのが一般的ですが、実はそれ以外に隠れフードロストというものがあります。
それは何かというと惣菜やファーストフード店で料理を提供する際に使用される食材を洗濯・加工する際の切れ端などが
廃棄につながってこれらが隠れフードロストと言われています。私が最近利用している吉野家の牛丼に乗っている玉葱の人気
があるようです。

実はあの玉葱はスーパーで売られている状態から茶色の皮を取り除いて、更に若干緑色の部分を落としています。
更に芯の部分もどうしても美味しくならないということで取り除くそうです。その部分はけっこうな量になって、オニオンリング
などに使えそうなくらい良いものだそうです。

意外だったのは、玉葱は人間しか食さないもので、家畜などの餌には向かないそうです。
それでこれを廃棄していたのですが、洗浄して砕いて乾燥させて熱を加えてパウダー状にするという動きがあるそうです。
まだ試作段階ではあるが、ボイラーを使って400度の熱を加えて乾燥させているそうです。
実際まだ一般には出てきてないのですが、パウダー状にしてふりかけにしたり、有名なポンパドウルのパンでは一部商品として
使用されているそうです。

課題としては、まだ開発段階ということとコストの面がまだ見えてきていないということです。
実際これをパウダー状にして味付けを変えて吉野家では使おうとしているみたいです。
今後フードロスというところでSDGsの取り組みとしては良いが、価格としてどのように影響してくるのかというところですね。
今まで廃棄するにもコストがかかっていたと思いますが、今後回収・洗浄・加工・乾燥するのにかかるコストがどこに吸収される
のかというところに目線を付け加えてみました。
```

### ほぼ宇宙旅行について

- 朝礼トピック

```
ＨＩＳが「ほぼ宇宙旅行」というものの予約をスタートしました。
このほぼ宇宙旅行行って何かというと、宇宙空間は高度１００キロ以上のところを指しますが、
この旅行では高度３０キロの成層圏のところまで行ける、ということで宇宙ではないけど、
宇宙の入口でほぼ宇宙ということで名前がつけられています。この場所でも青い地球を見ることが出来るそうです。

じゃあ実際何に乗って行くのかですが、気球型の宇宙船と言われるもので行くことが出来ます。
これはロケットではなく水素を使った気球なので、宇宙のい移動する初めてのカーボンニュートラルな方法と言われています。

この気球型宇宙船の特徴としては、まず１つ大きいこととして機内には重量があります。
なので、服装は自由で、機内で食事やバーカウンターがあってカクテル等を楽しむことが出来ます。

乗員は8名の旅行客1名パイロットが乗れます。ネットで写真を見ると、
機内は広々していておしゃれな空間になっています。そしてなんとWi-Fiも完備しているとのことで、
宇宙から友達や家族に連絡をしたり、今ですとＹＯＵＴＵＢＥで配信といったこともできるかもしれません。

実際の旅行時間ですが、2時間かけて高度３０キロまで上昇し、そこで2時間滞在し、また2時間かけて高度３０キロから地上に
戻ってくるという約6時間の旅行となります。

気になるお値段ですが、一人12万５０００ドルで日本円だと約1600万円だそうです。
すでに2024年の予約は完売していて、毎年１００便ほどのプライとが予定されていて
2028年には250便以上のフライトを予定しているそうです。
この値段だと正直お金持ちしか行ける希望がないですが、
この開発担当者は最終的には１００万円台で宇宙の入口に行けるようにしたいと話していて、
そのくらいの値段になると、旅行先として、宇宙の入口という選択肢が増えることで、
これから宇宙事業が活発になっていくのではないかなと思いました。

```
